# flutter_flavoring_demo

Flutter flavoring demo for android and IOS

## Getting Started

In this demo we have 3 flavors
 * Dev
 * Staging
 * Prod

## Run command
#### Dev
  flutter run --flavor dev
#### Staging
  flutter run --flavor staging
#### Prod
  flutter run --flavor prod
